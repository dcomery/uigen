# Task: Create Hello World Script

## Description
Create a standalone TypeScript hello world script at `src/scripts/hello-world.ts`. The script should print "Hello, World!" to stdout when executed. Use `console.log` for output. The script should be self-contained with no external dependencies beyond what the repo already provides.

## Files to Modify
- `src/scripts/hello-world.ts` — **create** new file

## Acceptance Criteria
- [ ] File `src/scripts/hello-world.ts` exists
- [ ] Running the script with `npx tsx src/scripts/hello-world.ts` prints exactly `Hello, World!` to stdout
- [ ] Script exits with code 0
- [ ] No lint errors when running `npx next lint`
